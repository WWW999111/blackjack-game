# Blackjack Card Game 🃏

基于 Python + Pygame 的多人在线 21 点（黑杰克）卡牌游戏。支持抢庄制、20+ 种道具、随机事件、悬赏系统、成就称号、牌背皮肤、桌布自定义等丰富玩法。

---

## 快速开始

### 环境要求

- Python 3.8 或更高版本
- pygame

```bash
pip install pygame
```

### 启动游戏

1. **启动服务端** — 双击 `start_game.bat`，或在终端运行：
   ```
   python server.py
   ```
2. **启动客户端** — 双击 `start_client.bat`，或在终端运行：
   ```
   python client_gui.py
   ```
3. 客户端连接后输入昵称，等待至少 **2 名玩家**加入
4. 所有人点击 **准备** 后游戏自动开始

### 单人模式

客户端也支持 **单人模式**，无需启动服务端即可与 AI 庄家对战。在主菜单选择「单人模式」进入。

## 游戏规则

### 基础 21 点玩法

| 操作 | 说明 |
|------|------|
| **要牌 (Hit)** | 从牌堆抽取一张牌，增加手牌点数 |
| **停牌 (Stand)** | 保持当前点数，结束本手操作 |
| **加倍 (Double)** | 仅限初始两张牌时可用，下注翻倍并只抽一张牌 |
| **分牌 (Split)** | 初始两张牌点数相同时可分成两手牌分别操作 |
| **投降 (Surrender)** | 初始两张牌时可放弃本手，返还一半下注筹码 |
| **保险 (Insurance)** | 庄家明牌为 A 时可购买，若庄家 BJ 则获赔 2 倍 |

### 点数计算

- 2~9 按面值计算，10/J/Q/K 为 10 点，A 可算 1 或 11 点
- **BlackJack**：初始两张牌为 A + 10/J/Q/K，赔率 **2.5 倍**
- **五龙 (Five Card Charlie)**：5 张牌仍未爆牌，直接获胜

### 抢庄制

每局开始前玩家可选择 **抢庄**。抢庄成功者成为本局庄家，与所有闲家对赌。无人抢庄则由系统担任庄家。

### 随机事件

每 3 局触发一次随机事件，改变本局规则：

| 事件 | 效果 |
|------|------|
| ⚡ 极速局 | 上限 17 点，庄家 13 点停牌，A+6 即为 BJ，超过 17 爆牌 |
| 💰 通货膨胀 | 本局所有人下注额强制翻倍 |
| 🃏 盲牌模式 | 所有人第二张牌为暗牌，看不见自己的点数 |
| 🩸 血月之夜 | 无点数上限，谁大谁赢，但抽到黑桃花色直接爆牌 |
| 🌪️ 龙卷风 | 发完初始牌后，所有人手牌顺时针平移一位 |
| 🏦 通货膨胀² | 赢家赔率 ×3，但爆牌者下一局禁止下注 |
| 🤫 道具禁用 | 本局所有主动道具无法使用 |
| ⚔️ 决斗时代 | 本局所有平局触发西部决斗 |

## 道具系统

道具在每局准备阶段选择携带。每局限带 **1 个主动道具 + 1 个被动道具**。

| 🎲 再来一次 | 800 | 玩家回合 | 突破规则强制额外摸一张牌 |
| 🃏 强买强卖 | 600 | 他人回合 | 强制一名对手下次必须要牌 |
| 🌫 致盲烟雾 | 400 | 他人回合 | 使一名对手看不见自己的点数 |
| 💣 搏命契约 | 300 | 下注阶段 | 本局输赢筹码 ×3.5 倍 |
| 🔀 移花接木 | 800 | 玩家回合 | 与任意玩家互换一张明牌 |
| ♻️ 回炉重造 | 600 | 玩家回合 | 扔掉初始两张牌，重新抽取 |
| 🎭 狸猫换太子 | 800 | 玩家回合 | 与庄家互换一张手牌 |
| 🧨 厄运信标 | 700 | 任意阶段 | 下一个要牌的人必抽到 10 点牌 |
| 🧤 第三只手 | 500 | 任意阶段 | 偷取一名玩家 1 个道具（无道具则偷 200 筹码） |
| 🧛 吸血印记 | 400 | 下注阶段 | 寄生一名玩家，他赢了分你 20% 利润 |
| 🤫 沉默干扰器 | 600 | 下注阶段 | 本局所有人主动道具失效 |
| 📦 命运盲盒 | 300 | 任意阶段 | 随机：2% 特等奖 / 10% 大奖 / 38% 保本 / 20% 谢谢参与 / 30% 诅咒 |

### 被动道具（自动触发）

| 道具 | 价格 | 触发条件 | 效果 |
|------|------|----------|------|
| ⏳ 时光倒流 | 1000 | 爆牌时 | 退回爆牌的牌，强制停牌 |
| 🛡 反弹镜像 | 600 | 被负面道具瞄准时 | 抵挡并反弹给使用者 |
| 📏 点数修正 | 800 | 结算时 | 若 ±1 点能赢，自动调整 |
| 🪬 金蝉脱壳 | 1000 | 输或爆牌时 | 本局不扣下注筹码 |
| 🔗 灵魂链接 | 500 | 开局自动 | 绑定筹码最多者，他赢你分红，他爆你连坐 |
| ⚖️ 天平倾斜 | 500 | 平局时 | 平局判你赢 |
| 💸 杀贫济富 | 300 | BJ + 筹码第一 | 强迫最穷玩家进贡 100 筹码 |

### 道具 Combo

- **时光倒流 + 再来一次** → 触发 **时空悖论**：爆牌时光倒流后，可用再来一次盲抽暗牌，若二次爆牌双倍扣注
- **反弹镜像** → 自动克制强买强卖、致盲烟雾等负面道具
- **灵魂链接 + 搏命契约** → 高风险高回报，绑定大佬后自己下搏命

## 悬赏系统

- **连胜触发**：连赢 3 局以上的玩家自动成为 **悬赏目标**
- **击败悬赏目标**可获得额外 500 筹码奖励
- 悬赏目标输掉后悬赏取消

## 外围下注

下注阶段可对其他玩家进行 **外围下注**：
- 押某玩家会 **爆牌**，命中赔 2 倍
- 押某玩家会 **赢庄家**，命中赔 2 倍

## 网络架构

```
┌─────────────────┐       TCP :5555       ┌─────────────────┐
│   client_gui    │ ◄──────────────────►  │     server       │
│  (Pygame 客户端) │                        │  (多线程服务端)   │
└────────┬────────┘                        └────────┬────────┘
         │                                          │
         │  支持多人同时连接                           │  游戏进行中加入→观战
         │  观战者下局自动加入                          │  下一局自动转为玩家
```

- 服务端监听端口 **5555**
- 服务端和客户端可在同一台电脑运行，也可局域网联机
- 支持 **观战模式**：游戏进行中加入的玩家自动观战，下一局自动参与

## 常见问题

**Q: 筹码输光了怎么办？**
筹码低于 100 时，可在准备阶段输入 `救济` 领取救济金（每日 3 次，每次 8888）。

**Q: 如何修改初始筹码？**
编辑 `server.py` 中的 `starting_chips` 默认值（1500）。

**Q: 支持几人游戏？**
理论上无上限，建议 2~6 人体验最佳。
