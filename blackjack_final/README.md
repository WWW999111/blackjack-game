# Blackjack Card Game 🃏

基于 Python 的多人在线 21 点（黑杰克）卡牌游戏，支持道具系统、随机事件、悬赏机制、成就称号等丰富玩法。

## 快速开始

### 环境要求
- Python 3.8+
- pygame

```bash
pip install pygame
```

### 启动方式

1. **启动服务端**：双击 `start_game.bat`（或运行 `python server.py`）
2. **启动客户端**：双击 `start_client.bat`（或运行 `python client_gui.py`）
3. 至少 2 名玩家加入后，所有玩家点击准备即可开始

## 游戏规则

### 基础玩法
- 标准 21 点规则：接近 21 点但不能超过
- **抢庄制**：每局开始前玩家可抢庄，庄家与所有闲家对赌
- BlackJack（A + 10/J/Q/K）赔付 **2.5 倍**
- 支持分牌、加倍、投降、保险
- **五龙**：5 张牌未爆直接获胜

### 特殊规则
- 每 3 局触发一次**随机事件**，影响本局规则
- 连赢 3 局以上成为**悬赏目标**，击败悬赏目标可获额外奖励
- 筹码低于 100 可领取**救济金**（每日 3 次，每次 8888）

## 道具系统

### 主动道具
| 道具 | 效果 |
|------|------|
| 透视眼镜 | 查看庄家暗牌 |
| 再来一次 | 额外摸一张牌 |
| 强买强卖 | 强制对手必须要牌 |
| 致盲烟雾 | 让对手看不见自己的点数 |
| 命运盲盒 | 随机获得奖励或惩罚 |
| 回炉重造 | 扔掉初始牌重新抽取 |
| 狸猫换太子 | 与庄家互换一张牌 |
| 搏命契约 | 本局输赢 ×3.5 倍 |
| 灵魂链接 | 绑定一名玩家共享输赢 |

### 被动道具
包括时光倒流、金蝉脱壳、反弹镜像、点数修正、厄运信标、吸血印记等。

## 项目结构

```
├── server.py          # 游戏服务端（处理游戏逻辑、网络通信）
├── client_gui.py      # 游戏客户端（Pygame 图形界面）
├── game_logic.py      # 21点核心规则（点数计算、分牌、BJ判定等）
├── Items.py           # 道具、成就、称号、牌背、桌布定义
├── Shop.py            # 商店系统、存档管理
├── Effects.py         # 画面特效
├── PNG/               # 扑克牌面图片
├── Card Back Visual/  # 牌背皮肤图片
├── start_game.bat     # 服务端启动脚本
├── start_client.bat   # 客户端启动脚本
├── 倍率速查表.md       # 倍率参考
└── 游戏规则.md         # 详细规则说明
```

## 自定义音乐

游戏支持背景音乐，对应各个界面的播放曲目：

| 文件名 | 播放场景 |
|--------|----------|
| `1 Main Theme.mp3` | 大厅 / 主菜单 |
| `2 Shop Theme.mp3` | 下注阶段 |
| `3 Tarot Pack Theme.mp3` | 抢庄阶段 |
| `4 Planet Pack Theme.mp3` | 游戏进行中 |
| `5 Boss Blind Theme.mp3` | 结算阶段 |

**添加方法：**

1. 在项目根目录新建 `Soundtrack` 文件夹
2. 将你喜欢的 MP3 文件按上表重命名后放入文件夹
3. 重启游戏即可生效

> 没有 `Soundtrack` 文件夹游戏也能正常运行，只是没有背景音乐。

## 网络架构

- 服务端监听端口 `5555`
- 客户端通过 TCP 连接，支持多人在线
- 游戏中加入的玩家自动进入**观战模式**，下一局自动加入
