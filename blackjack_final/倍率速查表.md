## 三、随机事件叠加倍率

### 💰 通货膨胀（下注强制翻倍）

| 结果 | 实际下注 | 拿回 | 净赢 |
|------|---------|------|------|
| 普通赢 | 200 | 400 | +200 |
| 普通赢 + 搏命 | 200 | 700 | +500 |
| 输 | 200 | 0 | -200 |
| 输 + 搏命 | 200 | 0，额外扣500 | -700 |

### 🏦 通货膨胀²（赢家×3赔率）

| 结果 | 拿回 | 净赢倍率 |
|------|------|---------|
| 普通赢 | 400 | **×3**（直接×3赔率） |
| BJ | 400 | **×3**（BJ赔率统一为×3） |
| 五龙 | 400 | **×3** |
| 搏命 + 通胀² | 700 | **×6**（3×搏命系数叠乘？❌见下方说明） |
| 输且爆牌 | 0，下局禁止下注 | -×1 |

> ⚠️ 通货膨胀²和搏命契约同时生效时：两者各自独立计算，取**更高者**而非相乘（防止无限叠加），即最终×max(3.0, 3.5)=**×3.5**

### ⚡ 极速局（Turbo规则，全局平移-4）

| 规则维度 | 普通局 | 极速局 |
|------|------|------|
| 爆牌线 | >21 | **>17** |
| 庄家停牌线 | ≥17（软17继续） | **≥13** |
| BlackJack | A+10点=21 | **A+6点=17** |
| 玩家绿色安全区 | =21 | **=17** |

- 玩家拿到A时非常纠结——A算11只剩6点空间，随时爆牌
- 庄家13点停牌，14-17区间玩家面临"停牌等庄爆/博小牌"的压力
- 10点牌（T）恐惧感极大提升

### 🩸 血月之夜（无上限，黑桃即爆）

| 结果 | 判定 |
|------|------|
| 持有任意黑桃牌 | 直接爆牌（不论点数） |
| 无黑桃，点数最大者 | 获胜，赔率×1 |
| 平局 | 正常平局规则 |

### ⚔️ 决斗时代 / 西部决斗（平局触发）

| 结果 | 判定 |
|------|------|
| 决斗牌更大 | 赢得对方所有赌注（含加倍等） |
| 决斗牌更小 | 输掉所有赌注 |
| 决斗也平 | 退还本金（极小概率） |

## 五、倍率优先级与豁免规则

```
优先级（从高到低）：
1. 爆牌判定（事件规则） → 先于一切结算
2. 五龙判定             → 5张不爆直接赢×2
3. BJ判定               → 赢×1.5（双方BJ平局）
4. 搏命契约             → ×3.5（豁免道具惩罚）
5. 道具惩罚             → ×0.5（仅普通赢时生效）
6. 通货膨胀²赔率        → ×3（与搏命取较大值）
7. 金蝉脱壳             → 退还本金（输/爆时生效）
8. 点数修正             → ±1点（比点数前触发）
9. 天平倾斜             → 平局变赢（赔率×1）
10. 吸血印记/灵魂链接   → 最后从盈利里抽成
```
